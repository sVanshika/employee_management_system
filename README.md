EMPLOYEE MANAGEMENT SYSTEM - SPRING MVC PROJECT

MODULES:
1. Employee Module
   1. Add employee
   2. Search employee
   3. Modify employee
   4. Delete employee
   
2. Loan Module
   1. Apply loan for an employee
   2. Modify loan
   3. Search loan
   
FEATURES:
1. Spring security
2. Transaction management
3. Hibernate ORM
4. JPA support
